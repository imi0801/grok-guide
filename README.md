## SuperGrok 开通流程新手版

整个流程只有四步：买卡密 → 验证卡密 → 填账号 ID → 确认充值。

### 第一步：用微信买一张充值卡密

打开 [GoPlus 充值入口](https://www.goplus.pro/)，在套餐列表里找到 Grok / SuperGrok，选好后用微信付款。付款完成后订单里会显示一串卡密，把它保存好。

![Grok Super 充值套餐购买页面](https://pic1.imgdb.cn/item/6a1cfaa56aa5b6be4f5f7d56.png)

### 第二步：验证卡密

进入充值页面，把刚才保存的卡密粘贴进输入框，点击验证卡密。

![Grok Super 充值卡密验证页面](https://pic1.imgdb.cn/item/6a1cfb0d6aa5b6be4f5f7d6f.png)

验证通过后，页面会让你填账号 ID。

![Grok 充值页面输入账号 ID](https://pic1.imgdb.cn/item/6a1cfb696aa5b6be4f5f7d81.png)

### 第三步：拿到自己的账号 ID

用浏览器登录你要开通会员的 Grok / X 账号，然后访问：`https://grok.com/api/auth/session`。

页面返回的信息里就有账号 ID；也可以点击充值页面里的超链接直接获取——前提同样是浏览器里已经登录了目标账号。

![Grok 获取账号 ID 页面](https://pic1.imgdb.cn/item/6a342b1991b65c4475ae63aa.png)

### 第四步：确认账号并充值

把账号 ID 粘贴回充值页面，点击确认账号信息。核对确认页显示的账号无误后，点击确认充值。

![Grok Super 充值最终确认页面](https://pic1.imgdb.cn/item/6a1cfd9b6aa5b6be4f5f7e31.png)

一般等待 2-8 分钟，回到 Grok 或 X 页面刷新，就能看到会员状态生效。

## 第一次开通前的准备

新手最容易栽在准备环节，开始前把这几件事做好：

- 确认 X / Grok 账号能正常登录，网络能正常打开 grok.com；
- 想清楚要开的是哪个套餐：Grok、Grok Super 还是 SuperGrok，名称随官方调整会有变化，以购买页为准；
- 浏览器里如果登录过多个账号，先退出无关账号，只保留要充值的那个；
- 付款后把订单号和卡密截图存好。

## 安全说明

涉及账号和钱的事，统一在这里说明白：

- 卡密充值属于第三方服务，不是 xAI / X 官方渠道，下单前自行核对套餐和订单；
- 正常流程从头到尾不会用到你的账号密码——只需要账号 ID。遇到索要密码的情况直接放弃；
- 账号 ID 只用于充值页面确认账号归属，不要随意发给别人；
- 建议开在自己的原账号上，不要买共享账号：多人混用一个账号，隐私和稳定性都没有保障；
- 保留订单记录，出现到账延迟时先联系售后，不要重复下单。

## SuperGrok 开通常见问题

### 1. SuperGrok 官方价格和卡密价格一样吗？

不完全一样。官方订阅是 $30/月 左右的美元定价，卡密是人民币标价，两者会有汇率和渠道差异。下单前对照 [GoPlus 购买页](https://www.goplus.pro/)的实际标价即可。

### 2. 国内没有海外信用卡，能开 SuperGrok 吗？

能。这正是卡密方式解决的问题：用微信付款买卡密，再用账号 ID 把会员充到自己账号，全程不需要绑任何海外卡。

### 3. 用微信充值 SuperGrok 会影响账号安全吗？

按本文流程，会员是开在你自己的原账号上、走正规通道开通的，目前没有出现普遍性的封号情况。不过第三方渠道没有人能打包票零风险，介意的话可以先充一个月观察。真正风险高的是共享账号——多人多地登录同一账号，最容易触发风控。

### 4. 买了卡密没用上，可以退款吗？

卡密未使用的情况下，带着订单号联系售后处理；会员已经充值到账的订单一般不退。付款记录和卡密截图都留好，沟通起来会快很多。

### 5. 免费额度够用的话还要开 SuperGrok 吗？

不用急着开。偶尔问答、轻度尝鲜，免费版完全够。等你发现自己每天都在做实时搜索、追热点、传文件让 Grok 分析，额度频繁见底，再升级也不迟。

### 6. 开通流程需要提供账号密码吗？

不需要。整个流程只用账号 ID 确认充值对象，密码始终在你自己手里。

### 7. 账号 ID 在哪里拿？打不开怎么办？

登录目标账号后访问 `https://grok.com/api/auth/session` 就能看到。打不开时先检查登录状态，换个浏览器再试，或者用充值页面里自带的获取链接。

### 8. 账号 ID 填错了怎么补救？

还没点确认充值的话，重新获取一遍再填即可。已经提交的，不要再下第二单，马上带着订单号和正确的账号 ID 找售后。

### 9. Grok Super 和 SuperGrok 是同一个东西吗？

搜索时两种叫法都常见，具体套餐名称和权益以 grok.com 和购买页当时的展示为准。下单前看清套餐名，别只看价格。

### 10. 充值成功后在哪里确认？

回到 Grok 或 X 页面刷新，看会员标识是否出现。几分钟内没变化可以退出重新登录；长时间未到账就联系售后查订单。

### 11. SuperGrok 和 ChatGPT Plus、Claude Pro 选哪个？

三者定位不同：SuperGrok 强在实时信息和 X 生态内容，ChatGPT Plus 综合能力全面，Claude Pro 擅长长文档和写作。经常刷热点、做社媒分析的人，SuperGrok 的优先级可以放高一些。
